DrmRemote
=========

DrmRemote 是一種透過 Microsoft® .NET Remoting 公開服務的 Web 應用程式，可用於不同 RMS 服務之間的通訊。DrmRemote 也能讓開發人員撰寫高效能應用程式，透過遠端處理項目使用伺服器對伺服器通訊。

DrmRemote 可在 RMS 根憑證伺服器或叢集以及任何授權伺服器或叢集上執行。其虛擬目錄對應於公開 .NET Remoting 服務的 RMS 服務。
