RMS 管理服務
============

管理服務會在 RMS 根叢集以及任何僅進行授權的叢集上執行。管理服務主控管理網站，並可讓您管理 RMS。

管理服務應用程式檔案 Default.aspx 位於 Admin 虛擬目錄 \[*RMS Web Site*\\\_wmcs\\Admin\] 中。其中 *RMS\_Web\_site* 應以提供 RMS 的網站名稱代替。
