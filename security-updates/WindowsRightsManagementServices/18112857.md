RMS 電腦憑證
============

RMS 電腦憑證可識別受 RMS 系統信任的電腦或裝置。

在用戶端啟動過程中，電腦或裝置會產生一個 RMS 電腦憑證。

RMS 電腦憑證包含所啟動電腦的公開金鑰。它由 Microsoft Activation Key 簽署，包含在 Service Pack 1 的 RMS 用戶端中。

電腦不需要成為 Active Directory 網域成員，就能收到 Lockbox 或電腦憑證。
