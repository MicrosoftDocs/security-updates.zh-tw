規劃
====

規劃包括瞭解您的重要資產，透過風險評估程序辨識它們面對的風險與弱點，然後建立企業安全性政策以管理風險。

這個學習捷徑的內容仍在持續發展當中。 [請讓我們知道](https://support.microsoft.com/common/survey.aspx?scid=sw;en;1257&showpage=1&ws=technet&sd=tech)您對哪些主題與內容類型感興趣。
