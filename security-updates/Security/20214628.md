IAS 的安全性指導
================

IAS 是 Microsoft 的「遠端驗證撥號使用者服務 (RADIUS)」伺服器的實作，IAS 會執行集中式連線驗證、授權，並且使用許多種網路存取，包括無線與虛擬私人網路 (VPN) 連線。 尋找 IAS 如何幫助讓公司網路更安全的資訊。
